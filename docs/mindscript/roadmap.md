# MindScript Roadmap

## v0.1.0

* Initial white paper release

## v0.2.0

* Add example specifications
* Validation rules for acceptance criteria
* Community feedback integration

## v0.3.0

* Introduce Context + Turn contracts (infrastructure layer)
* TypeScript bindings + validators package
* Initial verifier stdlib
* Template + librarian design

## v0.4.0

* CLI tooling: `mindscript validate|lock|verify|diff`
* JSON Schema publication from TS bindings
* Golden tests for core intents
* Canary + rollback template promotion

## v1.0.0

* Stable specification (spec + infrastructure unified)
* Reference implementations in multiple languages
* Full verifier stdlib hardened
* Community-contributed template registry
* Widespread adoption across domains
